# git-version-control-practice
A beginner-friendly repository demonstrating Git and GitHub workflows including commits, push, pull, branching, and version control best practices.

## Topics Covered

- Git initialization
- Tracking files
- Git status
- Staging changes
- Creating commits
- Pushing to GitHub
- Pulling changes
- Branching
- Merging
- Best practices

```bash
git add .

git push
```

## Pull Latest Changes
